Beadli

Beadli is a MERN-stack app that allows users to create, share, and browse pixel designs intended for fuse bead crafting. Developed with three others, my role in the project was front-end development using React (notably the interactive "paint" functionality and client-side authentication routing), UI/UX design, and floating back-end development. This app uses React to handle the view and functionality, Express and Node for server-side API routing, MongoDB for data storage, and Passport.js for authentication. This application follows the MVC design pattern. Built with React, MongoDB, Express, Node, Mongoose, Passport.js, Bitmap.js, JavaScript, Bootstrap, React Bootstrap, CSS3, and HTML5.
